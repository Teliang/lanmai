# lanmai
lanmai is a key mapping software in Linux.

# mapping keys
```
CapsLock => Left Ctrl
Sapce    => Space
Space h  => Left
Space j  => Down
Space k  => Up
Space l  => Right
```

# dependencies
+ libevdev
+ libudev

# build and run
```
make
sudo ./lanmai
```
